This full stack web application is to show, add, update and delete favorite songs,
while also allowing the user to form opinions about them.

There are also some end points to the API of the web app:

  • api/fav-list → To show a list of all the favorite songs and create new ones.
  • api/opinion-list → To show a list of all the opinions and create new ones.
  • api/fav-detail/<int:pk> → To show details about a specific song, update and delete it.
  • api/opinion-detail/<int:pk> → To show details about a specific opinion, update and delete it.
